OscController

A library by Tim Pulver, Paul Vollmer for the Processing programming environment.
Last update, 04/03/2013.


OscControllerKit simplifies the usage of certain controllers in your Processing projects. It depends on OSCulator, a Mac OS X application, which forwards controller data via OSC (Open Sound Control). OscContollerKit uses the OSC signals sent by OSCulator and adds an event layer similar to the Processing syntax.

Download

Download OscController version 0.1.1 (1) in .zip format.

Installation

Unzip and put the extracted OscController folder into the libraries folder of your Processing sketches. Reference and examples are included in the OscController folder.
